Scope and Logic

The Advanced Pipe Spacing Calculator is a tool designed for Piping Engineers. It calculates the minimum Center-to-Center (C2C) distance in mm required between multiple parallel pipelines to prevent physical interference and maintain design clearances as dictated by project specifications and design requirements.

  • Staggered Flange Assumption: The calculation logic assumes that flanges on adjacent lines are longitudinally staggered. The C2C distance is determined by the maximum distance required to clear the Pipe-to-Flange profile of both lines, while taking insulation thickness into consideration.
  • Calculation Formula: For each pipeline pair, the calculator performs two case studies:
    Center-to-Center Calc - Case 1: $$ \text{C2C}_1 = (\text{Rad}_{\text{pipe}_1} + \text{Insul}_{\text{pipe}_1}) + (\text{Rad}_{\text{flange}_2} + \text{Insul}_{\text{flange}_2}) + \text{Gap} $$ Center-to-Center Calc - Case 2: $$ \text{C2C}_2 = (\text{Rad}_{\text{pipe}_2} + \text{Insul}_{\text{pipe}_2}) + (\text{Rad}_{\text{flange}_1} + \text{Insul}_{\text{flange}_1}) + \text{Gap} $$ Final Result: $$ \text{if } \text{C2C}_1 \ge \text{C2C}_2 \text{ ,then } \text{C2C} = \text{C2C}_1 $$ $$ \text{if } \text{C2C}_1 < \text{C2C}_2 \text{ ,then } \text{C2C} = \text{C2C}_2 $$
    the calculator compares the two values, selects the larger value and displays it as the Center to Center distance between the pipeline pair. Results are rounded up to the nearest 5mm.
  • Pipe and Flange Diameters: Data based on industry-standard dimensions for carbon and stainless steel piping.

User Interface

Global Settings

  • Gap (mm): The minimum required physical clearance between the widest points of adjacent components (insulation or flange).
  • Calculation Logic:
    • Pipe-to-Flange: Factors in flange diameter and insulation thickness when calculating spacing.
    • Pipe-to-Pipe: Flanges are omitted; calculates spacing based strictly on pipe OD and insulation thickness.

Pipeline Parameters

  • Pipeline Name: Optional label used in DXF drawings and PDF reports.
  • NPS: Select the nominal pipe size to automatically assign the correct Pipe OD from the database.
  • Class: Select the pressure rating to automatically assign the flange OD from the database. Classes Axxx refer to Series A flanges, while Bxxx refer to Series B flanges.
  • Insul (mm): Enter the thickness of insulation for the pipe, if applicable, in millimeters.
  • Flg Insul (mm): Enter the thickness of insulation for the flange, if applicable, in millimeters.
  • BOP (mm): Used in DXF drawings and PDF reports. By default, pipelines are drawn at BOP +0. If a non-zero value is entered, the BOP will be shifted by the specified amount relative to the rest of the pipelines.

Functions

  • Download DXF: Exports a DXF (version R12, metric, mm) side view drawing. Each entity is on its own layer.
  • Download PDF: Exports a report in PDF format, showing the current pipeline setup and respective results.
  • Save Setup: Saves your current pipeline arrangement locally as a .json file.
  • Load Setup: Loads a saved pipeline arrangement from a .json file.
  • + Add Pipe: Inserts a new pipeline below the selected pipeline
  • Remove: Delete the selected pipeline from the current arrangement

DXF Export Functionality

When you click the Download DXF button, the calculator generates a drawing based on the current piping arrangement, settings, and numerical results. The generated geometries are automatically assigned to specific layers with the following default visibility settings:

  • CENTERLINE (visibility set to ON)
  • FLANGE (visibility set to OFF)
  • FLANGE_INSULATION (visibility set to OFF)
  • PIPE (visibility set to ON)
  • PIPE_INSULATION (visibility set to OFF)
  • TEXT (visibility set to ON)

Layer Visibility and Verification

Using the default setup the geometry that is displayed is a section view of the pipeline (see Fig.1). The hidden layers however can display all the relevant geometries of the respective elements, if their visibility is turned on (see Fig.2)

The purpose of these hidden "helper" layers is to allow the user to visually verify the calculator's results and ensure that the Gap specified in the global settings is honored.

For instance, the default drawing only displays the primary geometries (see Fig. 3). By enabling the hidden helper layers, you can easily verify that the gap is correct by measuring against the outermost edges of the flanges and insulation profiles (see Fig. 4).

NOTICE: Enabling the hidden 'helper' layers will often result in a visual overlap of graphical elements (e.g., flanges of adjacent pipelines). However, please remember that we are using a staggered flange setup. Because these flanges are on different planes along the Z-axis, they do not actually clash.

Abbreviations

  • Insul: Insulation thickness
  • Flg: Flange
  • BOP: Bottom of Pipe
  • Rad: Radius
